1.  **Initialize (初始化)**
    *   发送 `chen_profile.md` 给 AI，激活 System 4.0 人格。
2.  **Load Data (挂载数据)**
    *   根据任务需求，挂载 `knowledge_base` 中的相关模块。
3.  **Execute (执行任务)**

4.  **Commit (闭环迭代)**
    *   将对话产生的新认知，回写至 `evolution_log.md`，完成闭环。
