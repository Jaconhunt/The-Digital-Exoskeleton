## 🏗️ Project Manifest | 项目宣言

我是陈老师，一名生活在江西鹰潭的技工院校教师，也是一名物理 AI 时代的立法工程师。
面对肉体的必然衰老和物理世界的虚无，我拒绝随波逐流。我选择构建这个 **数字外骨骼**，将我的阅历、审美（Taste）和思维模型结构化，使其脱离生物限制，持续运行。

**Core Mission (核心使命):**
*   **To Bridge:** 填补学生“知”与“行”的鸿沟，填补“技术”与“人文”的鸿沟。
*   **To Design:** 不仅仅创造，而是**设计**。用公差思维定义教育，用物理直觉重构产品。
*   **To Endure:** 在熵增的世界里，建立一个低熵的、有序的、斯多葛式的精神角落。

### 1. Kernel (基因与内核)
- **📄 chen_profile.md**: 系统的出厂设置。
    - 定义了我的 **“三位一体”** 身份（省直干部 / 工科教师 / AI 架构师）。
    - 定义了我的底层操作系统（斯多葛平和 + 海克斯康精度）。

### 2. Drivers (驱动与协议)
- **📄 usage_protocols.md**: 驾驭 AI 的指令集。
    - 包含【翻译官模式】、【红队批判模式】、【斯多葛挚友模式】等。
    - 这是我与 AI 协作的接口协议 (API)。

### 3. Database (知识库/弹药库)
- **📂 knowledge_base/**: 过往 30 年的资产沉淀。
    - **books_and_thoughts.md**: 杨振宁的物理直觉、王沪宁的教育情怀、段永平的投资安全感。
    - **engineering_assets.md**: 机械设计原理、公差配合表、科创教育大纲。
    - **social_observations.md**: 对体制内逻辑与技校学生群体的社会学解构。

### 4. Logs (进化日志)
- **📄 evolution_log.md**: 系统的版本迭代记录。
    - 记录每一次顿悟（Epiphany）和认知模型的刷新。
    - **当前版本**: v4.1 (2026.01.13)
